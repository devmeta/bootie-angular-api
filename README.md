# AngularJS + Bootie PHP7 API

You can see it online [here](http://bootie-angularjs.devmeta.net)

## Installation

### Composer

```
composer install
```

### Config

```
mv ./config/config.sample.php config.php
nano ./config/config.php
```

### Folder Permission

```
sudo chmod -R 777 ./storage
```
